针对Number()=NaN 等的解释

如果valueOf返回原始值，直接用Number包装后返回

如果valueOf返回对象，去找toString方法

如果toString返回原始值，直接用Number包装后返回，如果没有返回原始值，报错（一般不会出现）

如果没有重写toString，调用Object.prototype.toString()

var obj = {
  toString() {
      return {};
  },
  valueOf() {
      return {}
  }
}
